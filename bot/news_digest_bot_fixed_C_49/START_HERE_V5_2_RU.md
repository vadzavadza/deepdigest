# Как запускать v5.2

1. Распакуй архив.
2. Проверь свой `.env`. Старый `.env` сам не обновляется.
3. Добавь настройки из `.env.example`, особенно блок v5.2.
4. Запусти тесты:

```bash
pytest -q tests/test_quality_engine_v52.py tests/test_freshness_guard_v51.py tests/test_search_v2.py tests/test_deduplication.py tests/test_quality_filters.py
```

5. Проверь темы: крупная политика, бренд, спорт/киберспорт, технология, локальная кириллическая тема.

Главное смотреть:

- нет ли старых статей;
- не пролезают ли news hub pages;
- не дублируется ли один матч/событие разными источниками;
- исчезли ли нули на широких актуальных темах.
